Basics

IMPORT

When you need to reference another file or resource.

@import url('http://www.yoursite.com/');

@import url('https://fonts.googleapis.com');

Universal Selector

Use this to create universal rules to your site. Keep it at the top.

* { paramaters go here }

* {
margin: 0;
}

Custom Properties

Use these to create variables to reference later.

:root { --property: val; }

value has to match the parameters of desired element.

:root {
--primary-clr: #456;
}

Adjacent Sibling

A combinator used when editing elements in a group.

.selector + .selector
.selector > .selector

Use this when editing elements that are after or behind a specific class without extra markup.

Use this on child elements.
