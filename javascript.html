Variables

Variables are the basic form of data storage.

let name = "John"; // String :iteral
let age = 30; // Numberal Literal
let isApproved = true; // Boolean Literal
let middleName = undefined;
let lastName = null; // Clears variable.

How to compressed everything into a single variable.

let person = { name = "John", age = 23, occupation = Gundam Pilot, }

They cannot be a reserved keyword.
Should be meaningful.
Cannot start with a number. (1name)
Cannot contain a space or hyphen (-)

Constants

You can also use 'const' to create a constant.
Constants are used when you don't need to have a variable changed.

const age = 30;

NOTE: DO NOT USE VAR. Use let as it has a more limited scope.

Notation

To notate, do this:

person.occupation;

Loops

Loops are used to create a series of statements and funtions.

For Loop

for (let i = 0; i < 5; i++) { console.log('Hello World!'); if (i % 2 !== 0) console.log(i); }

i will start at 0 and add 1 until it reaches 5.

In each loop the if statement will run.

While Loop

while (let a < 10) { text += "The number is " + a; a++ }

While a is less than 10, the text will be printed.
