Variables

Variables are the basic form of data storage.

let name = "John"; // String :iteral
let age = 30; // Numberal Literal
let isApproved = true; // Boolean Literal
let middleName = undefined;
let lastName = null; // Clears variable.

They cannot be a reserved keyword.
Should be meaningful.
Cannot start with a number. (1name)
Cannot contain a space or hyphen (-)

Constants

You can also use 'const' to create a constant.
Constants are used when you don't need to have a variable changed.

const age = 30;

NOTE: DO NOT USE VAR. Use let as it has a more limited scope.
